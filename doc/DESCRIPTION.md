Lingua 90 is a self-hosted, offline-friendly language-learning course.

* A 90-day journey with daily vocabulary flash-cards, a captioned video lesson,
  and a reading exercise (read-aloud, sentence-by-sentence, and an online
  synchronised-translation mode).
* English and Chinese learning tracks; English / French / Chinese interface.
* **Per-user progress and settings** stored on the server in one small JSON file
  per learner — no database. When the app is behind the YunoHost SSO, each
  learner's progress is tied to their YunoHost account automatically; otherwise
  a shared "Who's learning?" profile picker is shown.
